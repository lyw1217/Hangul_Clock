## 사용 재료
  1. 멀바우 18T 집성목
  2. 5T 형광연두(반투명) 아크릴
  3. 3T 광확산 PC

## 사용 부품
  1. Arduino Uno
  2. DS3231 RTC
  3. XL4015 DC-DC 가변 컨버티
  4. QN16-C1 16파이 푸쉬 버튼 스위치
  5. 16V 1000uF 커패시터
  6. 12V 1.5A
